## Spring Boot配置

### application.yml 配置示例

```yaml

  datasource:
    driver-class-name: com.mysql.cj.jdbc.Driver
    url: jdbc:mysql://localhost:3306/auction?useUnicode=true&characterEncoding=utf8&serverTimezone=Asia/Shanghai
    username: root
    password: your_password

  # 可选：自动执行SQL（开发环境）
  sql:
    init:
      mode: never  # never/always/embedded
      schema-locations:
        - classpath:schema_complete.sql
      data-locations:
        - classpath:data_init.sql

    map-underscore-to-camel-case: true
  global-config:

      logic-delete-field: deleted

      logic-not-delete-value: 0
```

## 常见问题排查

### 问题1：外键约束失败

**错误信息：**
```
ERROR 1215 (HY000): Cannot add foreign key constraint
```

**解决方案：**
1. 确保引用的表已存在
2. 检查字段类型是否一致
3. 检查字符集是否一致

```sql
-- 检查外键约束
SELECT
    CONSTRAINT_NAME,
    TABLE_NAME,
    REFERENCED_TABLE_NAME
FROM information_schema.KEY_COLUMN_USAGE
WHERE TABLE_SCHEMA = 'auction'
AND REFERENCED_TABLE_NAME IS NOT NULL;
```

### 问题2：字符集问题

**症状：** 中文乱码

**解决方案：**
```sql
-- 修改数据库字符集
ALTER DATABASE auction CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

-- 修改表字符集
ALTER TABLE users CONVERT TO CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

### 问题3：乐观锁版本号冲突

**错误信息：**
```
OptimisticLockException
```

**解决方案：**
- 确保实体类添加了`@Version`注解
- 检查数据库中version字段是否存在
- version字段初始值应为0

### 问题4：登录失败

**可能原因：**
1. 密码加密方式不一致（当前使用BCrypt）
2. 用户状态为LOCKED或DISABLED

**检查SQL：**
```sql
-- 检查用户状态
SELECT username, status FROM users WHERE username = 'admin';

-- 重置用户密码
UPDATE users
SET password = '$2a$10$N.zmdr9k7uOCQb376NoUnuTJ8iAt6Z5EHsM8lE9lBOsl7iAt6Z5EH',
    status = 'ACTIVE'
WHERE username = 'admin';
```


```sql
-- 清理30天前的登录日志
```

## 数据备份与恢复

```bash

mysqldump -u root -p --no-data auction > auction_schema.sql
```
