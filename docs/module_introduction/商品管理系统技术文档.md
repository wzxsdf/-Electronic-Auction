## 1. 系统概述

### 1.1 功能简介

商品管理模块是拍卖系统的基础模块，负责拍卖商品的信息管理和维护，包括：

- ✅ **商品创建**: 商家创建和管理拍卖商品
- ✅ **商品查询**: 用户和商家查询商品信息
- ✅ **商品更新**: 修改商品信息和状态
- ✅ **商品分类**: 商品分类和标签管理
- ✅ **图片管理**: 商品图片上传和管理
- ✅ **商品审核**: 商品发布前的审核机制

### 1.2 设计目标

- **易用性**: 简单直观的商品管理界面
- **完整性**: 确保商品信息的完整和准确
- **安全性**: 保护商家和用户的数据安全
- **扩展性**: 支持多种商品类型和属性
- **性能**: 高效的商品查询和展示

### 1.3 技术特点

| 特性 | 技术方案 | 优势 |
|------|----------|------|
| 数据存储 | MySQL | 关系型数据管理 |
| 图片存储 | 文件系统/OSS | 灵活的存储方案 |
| 数据验证 | Jakarta Validation | 统一的参数校验 |
| 缓存加速 | Redis缓存 | 提升查询性能 |
| 搜索功能 | 全文搜索 | 快速商品检索 |

## 3. 数据库设计

### 3.1 商品表 (products)

```sql
CREATE TABLE `products` (
  `id`                BIGINT AUTO_INCREMENT PRIMARY KEY COMMENT '商品ID',
  `name`              VARCHAR(200) NOT NULL COMMENT '商品名称',
  `description`       TEXT COMMENT '商品描述',
  `image_url`         VARCHAR(500) COMMENT '商品主图URL',
  `category_id`       BIGINT COMMENT '分类ID',
  `brand`             VARCHAR(100) COMMENT '品牌',
  `model`             VARCHAR(100) COMMENT '型号',
  `status`            VARCHAR(20) NOT NULL DEFAULT 'ACTIVE' COMMENT '商品状态',
  `creator_id`        BIGINT NOT NULL COMMENT '创建者ID',
  `view_count`        INT DEFAULT 0 COMMENT '浏览次数',
  `auction_count`     INT DEFAULT 0 COMMENT '拍卖次数',
  `created_at`        DATETIME DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
  `updated_at`        DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间',
  `version`           INT DEFAULT 0 COMMENT '乐观锁版本号',
  INDEX `idx_name` (`name`),
  INDEX `idx_category_id` (`category_id`),
  INDEX `idx_status` (`status`),
  INDEX `idx_creator_id` (`creator_id`),
  INDEX `idx_created_at` (`created_at`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='商品表';
```

**字段说明**:
- `id`: 主键，自增长
- `name`: 商品名称，必填，2-200字符
- `description`: 商品详细描述，支持富文本
- `image_url`: 商品主图URL
- `status`: 商品状态（ACTIVE, INACTIVE, DELETED, PENDING_REVIEW）
- `creator_id`: 创建者ID（商家）
- `view_count`: 商品浏览次数
- `auction_count`: 商品参与拍卖的次数

### 3.2 商品图片表 (product_images)

```sql
CREATE TABLE `product_images` (
```

## 4. 核心功能

### 4.1 商品创建

```java
/**
 * 创建商品
 */
public ProductResponse create(CreateProductRequest request) {
    // 1. 验证商品信息
    validateProductInfo(request);
    
    // 2. 创建商品实体
    Product product = new Product();
    product.setName(request.getName());
    product.setDescription(request.getDescription());
    product.setImageUrl(request.getImageUrl());
    product.setCategoryId(request.getCategoryId());
    product.setBrand(request.getBrand());
    product.setModel(request.getModel());
    product.setStatusEnum(ProductStatus.ACTIVE);
    product.setCreatorId(request.getCreatorId());
    
    // 3. 保存商品信息
    Product savedProduct = productRepository.save(product);
    
    // 4. 处理商品图片
    if (request.getImages() != null && !request.getImages().isEmpty()) {
        saveProductImages(savedProduct.getId(), request.getImages());
    }
    
    // 5. 缓存商品信息
    cacheProductInfo(savedProduct);
    
    // 6. 记录操作日志
    log.info("商品创建成功: productId={}, name={}", 
            savedProduct.getId(), savedProduct.getName());
    
    return convertToResponse(savedProduct);
}

/**
 * 验证商品信息
 */
private void validateProductInfo(CreateProductRequest request) {
    // 验证商品名称
    if (request.getName() == null || request.getName().length() < 2) {
        throw new BizException(ErrorCode.INVALID_PRODUCT_NAME, "商品名称至少2个字符");
    }
    
    // 验证图片URL
    if (request.getImageUrl() != null && !isValidImageUrl(request.getImageUrl())) {
        throw new BizException(ErrorCode.INVALID_IMAGE_URL, "图片URL格式不正确");
    }
    
    // 验证分类
    if (request.getCategoryId() != null && 
        !categoryRepository.existsById(request.getCategoryId())) {
        throw new BizException(ErrorCode.CATEGORY_NOT_FOUND, "分类不存在");
    }
}

/**
 * 保存商品图片
 */
private void saveProductImages(Long productId, List<ProductImageRequest> images) {
    images.forEach(imageRequest -> {
        ProductImage image = new ProductImage();
        image.setProductId(productId);
        image.setImageUrl(imageRequest.getImageUrl());
        image.setImageOrder(imageRequest.getOrder());
        image.setIsPrimary(imageRequest.getIsPrimary() != null && 
                           imageRequest.getIsPrimary());
        
        productImageRepository.save(image);
    });
}
```

```java
/**
 * 根据ID查询商品详情
 */
public ProductResponse getById(Long id) {
    // 1. 尝试从缓存获取
    Product product = getProductFromCache(id);
    
    // 2. 缓存不存在则从数据库查询
    if (product == null) {
        product = productRepository.findById(id);
        if (product == null) {
            throw new BizException(ErrorCode.PRODUCT_NOT_FOUND);
        }
        
        // 3. 缓存商品信息
        cacheProductInfo(product);
    }
    
    // 4. 增加浏览次数
    incrementViewCount(id);
    
    // 5. 加载商品图片
    List<ProductImage> images = productImageRepository.findByProductId(id);
    
    // 6. 转换为响应对象
    ProductResponse response = convertToResponse(product);
    response.setImages(convertImageResponses(images));
    
    return response;
}

/**
 * 查询所有商品

public List<ProductResponse> listAll() {

    return products.stream()

        .collect(Collectors.toList());

 * 根据分类查询商品
 */
public List<ProductResponse> getByCategory(Long categoryId) {
    List<Product> products = productRepository.findByCategoryId(categoryId);
    
    return products.stream()
        .map(this::convertToResponse)
        .collect(Collectors.toList());
}

/**
 * 搜索商品
 */
public List<ProductResponse> searchProducts(String keyword) {
    // 1. 构建搜索条件
    LambdaQueryWrapper<Product> wrapper = new LambdaQueryWrapper<>();
    wrapper.and(w -> w.like(Product::getName, keyword)
                      .or()
                      .like(Product::getDescription, keyword));
    wrapper.eq(Product::getStatus, ProductStatus.ACTIVE);
    
    // 2. 执行搜索
    List<Product> products = productRepository.findAll(wrapper);
    
    // 3. 转换结果
    return products.stream()
        .map(this::convertToResponse)
        .collect(Collectors.toList());
}
```

### 4.3 商品更新

```java
/**
 * 更新商品信息
 */
@Transactional(rollbackFor = Exception.class)
public ProductResponse update(Long id, UpdateProductRequest request) {
    // 1. 查询商品信息
    Product product = productRepository.findById(id);
    if (product == null) {
        throw new BizException(ErrorCode.PRODUCT_NOT_FOUND);
    }
    
    // 2. 验证权限
    if (!product.getCreatorId().equals(request.getOperatorId())) {
        throw new BizException(ErrorCode.FORBIDDEN, "无权修改此商品");
    }
    
    // 3. 更新商品信息
    if (request.getName() != null) {
        product.setName(request.getName());
    }
    if (request.getDescription() != null) {
        product.setDescription(request.getDescription());
    }
    if (request.getImageUrl() != null) {
        product.setImageUrl(request.getImageUrl());
    }
    if (request.getBrand() != null) {
        product.setBrand(request.getBrand());
    }
    if (request.getModel() != null) {
        product.setModel(request.getModel());
    }
    
    // 4. 保存更新
    Product updatedProduct = productRepository.save(product);
    
    // 5. 清除缓存
    clearProductCache(id);
    
    // 6. 记录操作日志
    log.info("商品更新成功: productId={}, operatorId={}", id, request.getOperatorId());
    
    return convertToResponse(updatedProduct);
}
```

```java
/**
 * 上传商品图片
 */
public String uploadImage(MultipartFile file, Long userId) {
    // 1. 验证文件
    validateImageFile(file);
    
    // 2. 生成文件名
    String originalFilename = file.getOriginalFilename();
    String extension = originalFilename.substring(originalFilename.lastIndexOf("."));
    String filename = "product_" + userId + "_" + System.currentTimeMillis() + extension;
    
    // 3. 保存文件
    String imageUrl;
    try {
        imageUrl = fileStorageService.saveFile(file, filename);
    } catch (Exception e) {
        throw new BizException(ErrorCode.IMAGE_UPLOAD_FAILED, "图片上传失败");
    }
    
    // 4. 记录操作日志
    log.info("图片上传成功: userId={}, filename={}", userId, filename);
    
    return imageUrl;
}

/**
 * 验证图片文件
 */
private void validateImageFile(MultipartFile file) {
    // 验证文件是否为空
    if (file == null || file.isEmpty()) {
        throw new BizException(ErrorCode.INVALID_FILE, "文件不能为空");
    }
    
    // 验证文件类型
    String contentType = file.getContentType();
    if (!isValidImageContentType(contentType)) {
        throw new BizException(ErrorCode.INVALID_FILE_TYPE, "只支持JPG、PNG、GIF格式的图片");
    }
    
    // 验证文件大小（最大5MB）
    if (file.getSize() > 5 * 1024 * 1024) {
        throw new BizException(ErrorCode.FILE_TOO_LARGE, "图片大小不能超过5MB");
    }
}

/**
 * 设置商品主图
 */
@Transactional(rollbackFor = Exception.class)
public void setPrimaryImage(Long productId, Long imageId, Long operatorId) {
    // 1. 验证权限
    Product product = productRepository.findById(productId);
    if (product == null || !product.getCreatorId().equals(operatorId)) {
        throw new BizException(ErrorCode.FORBIDDEN, "无权操作此商品");
    }
    
    // 2. 取消原主图
    productImageRepository.updatePrimaryByProductId(productId, false);
    
    // 3. 设置新主图
    ProductImage image = productImageRepository.findById(imageId);
    if (image != null && image.getProductId().equals(productId)) {
        image.setIsPrimary(true);
        productImageRepository.save(image);
        
        // 4. 更新商品主图URL
        product.setImageUrl(image.getImageUrl());
        productRepository.save(product);
        
        // 5. 清除缓存
        clearProductCache(productId);
        
        log.info("主图设置成功: productId={}, imageId={}", productId, imageId);
    }
}
```

| 权限验证 | 需要商家权限 | 无权创建商品 |

### 6.2 商品状态规则

| 状态 | 说明 | 允许操作 |
|------|------|----------|
| ACTIVE | 激活状态，可参与拍卖 | 创建拍卖、修改信息 |
| INACTIVE | 未激活，不可参与拍卖 | 修改信息、激活 |
| DELETED | 已删除，不可恢复 | 无 |
| PENDING_REVIEW | 待审核，等待审核 | 等待审核结果 |

### 6.3 图片管理规则

- **图片数量**: 每个商品最多10张图片
- **主图设置**: 每个商品只能有一张主图
- **图片排序**: 图片按排序序号显示
- **图片格式**: 仅支持JPG、PNG、GIF格式
- **图片大小**: 单张图片最大5MB

## 9. 常见问题

### Q1: 如何修改商品的主图？

**A**: 调用设置主图接口，或重新上传图片并设置为主图。

### Q2: 商品可以删除吗？

**A**: 商品逻辑删除，状态变为DELETED，数据不会真正删除。

### Q3: 如何批量导入商品？

**A**: 系统支持CSV文件批量导入，可联系管理员获取模板。

### Q4: 商品图片存储在哪里？

**A**: 图片可存储在本地文件系统或云存储（如OSS），通过配置切换。

### Q5: 如何优化商品搜索性能？

**A**: 使用Redis缓存热门商品，引入Elasticsearch实现全文搜索。
