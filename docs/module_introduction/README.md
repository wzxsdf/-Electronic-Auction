    ▼           ▼           ▼

     ▼            ▼            ▼

## 🔧 技术栈总览

### 后端技术
- **框架**: Spring Boot 3.x
- **数据库**: MySQL 8.0+
- **缓存**: Redis 6.0+
- **通讯**: WebSocket (STOMP)
- **安全**: Spring Security + JWT
- **ORM**: MyBatis-Plus

### 前端技术
- **框架**: 可支持任何现代前端框架
- **通讯**: WebSocket Client
- **状态管理**: Redux/Vuex

### 基础设施
- **部署**: Docker / 传统部署
- **监控**: Spring Boot Actuator
- **日志**: Logback
